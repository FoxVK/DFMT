Libdfmt_error libdfmt_cmd_status ( Libdfmt_tuner *  tuner,
char *  status_byte 
)

Gets status byte of tuner. Useful for determining is is safe to send next command.

Unlike libdfmt_check_bussy() this function does not return LIBDFMT_ERROR_TUNER_BUSY as result of status byte. This error code will be returned only in case unfinished previvous I2C read or write.

Parameters
tunerSelected tuner
status_bytePointer to store status byte.
Returns
LIBDFMT_OK on success or other Libdfmt_error.
